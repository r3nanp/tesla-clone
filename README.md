# Tesla Clone
  A clone of tesla website written in react, using framer motion to made animations.

## Techologies
- React
- Typescript
- Framer Motion
- styled-components

## How to run

```
# Clone repository

git clone https://github.com/r3nanp/tesla-clone.git
```

```
# At the root of the project, run:

yarn or npm install
```

```
# To run the app in your machine:

yarn start
```
